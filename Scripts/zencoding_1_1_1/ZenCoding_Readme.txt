### Installation

Place contents of zip file in the Scripts directory.

 ~/Library/Application Support/TextWrangler/Scripts/

To assign a keyboard shortcut for the script go to Window > Palettes > Scripts. Locate the Zen Coding script in the palette that comes up. Click the Set Key button and enter the keyboard shortcut you wish to use. It will tell you if the shortcut is already in use.
